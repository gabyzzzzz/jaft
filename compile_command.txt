For exe using jaft:

g++ -O2 -s main.cpp -L../lib -ljaft -static-libstdc++ -static-libgcc -mconsole -o build.exe
g++ -O2 -s sprite_editor.cpp -L../lib -ljaft -static-libstdc++ -static-libgcc -mconsole -o sprite_editor.exe

-O2 → Optimizes the binary 
-s → Strips debug symbols 

For lib:
g++ -O2 -s -c jaft.cpp -o jaft.o
ar rcs libjaft.a jaft.o
